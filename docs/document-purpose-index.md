## 정리 원칙

- 허가 판단은 `permit-readiness-internal-standard.md`에 통합한다.
- 예전 상담 준비, KTL 문의, remediation 제출 패키지는 기준 문서와 충돌 가능성이 있어 삭제했다.
- 새 문서가 생기면 이 인덱스에 추가하기 전에 현재 기준과 충돌하지 않는지 확인한다.
