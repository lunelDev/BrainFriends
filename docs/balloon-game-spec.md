# 풍선 키우기 기획/구현 정리

## 1. 개요

풍선 키우기는 `음량 유지 훈련`에 초점을 둔 게임이다.  
사용자는 마이크에 소리를 내서 풍선을 목표 크기까지 키워야 하며, 너무 큰 소리를 오래 유지하면 풍선이 터진다.

핵심 경험은 다음 3가지다.

- 소리를 내면 풍선이 즉시 커진다.
- 너무 크지 않은 안정적인 발성이 가장 유리하다.
- 제한 시간 안에 목표 크기에 도달하면 성공한다.

## 2. 게임 목표

사용자는 각 세션에서 일정 시간 동안 발성을 유지해 풍선을 `targetSize`까지 키운다.

성공 조건:

- 제한 시간 안에 풍선 크기가 목표치에 도달

실패 조건:

- 위험 구간 이상의 큰 소리를 일정 시간 이상 유지해 풍선이 터짐
- 제한 시간이 끝날 때까지 목표 크기에 도달하지 못함

결과 리포트에는 다음이 표시된다.

- 발성 시간
- 평균 음량
- 안정성
- 시간 초과 시 목표 도달률

- 초반 사용자 적응용

### 중

- 시작 크기: `16`
- 목표 크기: `100`

- 위험 시작: `84`
- 위험 유지 허용 시간: `350ms`
- 성장 속도: `15 / sec`

특징:

- 하보다 안전 구간이 좁다
- 성장량이 조금 감소
- 발성 조절 능력을 더 요구한다

### 상

- 제한 시간: `20초`
- 시작 크기: `14`

- 위험 시작: `82`
- 위험 유지 허용 시간: `300ms`
- 성장 속도: `13 / sec`

특징:

- 가장 좁은 안전 구간
- 위험 허용 시간이 가장 짧다
- 작은 실수도 실패로 이어질 가능성이 크다

## 4. 입력 방식

풍선 키우기는 STT가 아니라 `음량 기반` 게임이다.

관련 구현:

- [useBalloonAudioInput.ts](c:/Users/pc/Desktop/ProjectFiles/BrainFriends/src/lib/audio/useBalloonAudioInput.ts)

특징:

- `AudioContext + AnalyserNode` 기반
- 브라우저 마이크 입력을 실시간 분석
- 게임용 강도 값으로 보정된 수치를 사용
- 사용자가 말하는 동안 값이 빠르게 올라오도록 smoothing을 낮춘 상태

즉 화면의 `dB`는 절대 물리 dB라기보다, 게임용으로 환산된 입력 강도에 가깝다.

## 5. 라운드 구조

풍선 키우기는 문제 단위가 아니라 `1 세션 = 1 라운드` 구조다.

흐름:

1. 난이도 선택
2. 마이크 활성화
3. 발성 시작
4. 실시간으로 풍선 성장/위험 판정
5. 성공 또는 실패 결과 모달 표시

별도 문제 카드나 다음 문제 개념은 없다.

## 6. 성장 판정 방식

현재 구현은 `안전 구간 정확히 일치`보다 `소리가 들어오면 성장` 쪽으로 완화된 상태다.

핵심 분기:

- 안전 구간: 가장 안정적으로 성장
- 안전 구간 바로 아래: 일정한 톤이면 성장
- 일반 발성 구간: 위험 구간이 아니고 목소리가 들어오면 성장
- 위험 구간: 위험 누적 증가

즉 최근 구현 기준으로는 `소리를 냈을 때 바로 커지는 체감`을 우선한다.

## 7. 감소/유지 로직

현재 구현에서는 자동 감소를 제거했다.

즉:

- 발성을 멈춰도 자동으로 줄어들지 않음
- 다시 소리를 내면 이어서 성장 가능
- 실패는 `위험 구간 유지` 또는 `시간 초과`로만 발생

이건 사용자 피드백을 반영한 결정이다.

## 8. 위험 처리 방식

관련 상태:

- `dangerRatio`
- `dangerMsRef`

동작:

- 위험 구간 이상이면 위험 누적 증가
- 일정 시간 이상 계속 크면 `burst` 실패
- 위험 구간을 벗어나면 누적 초기화

즉 사용자는 단순히 크게 소리 내는 것이 아니라, 너무 큰 소리를 오래 유지하지 않는 조절도 필요하다.

## 9. 점수 측정 방식

풍선 키우기의 결과는 전통적인 점수보다 `세션 품질` 지표에 가깝다.

표시 값:

- `voicedSeconds`: 실제 발성 시간
- `averageVolume`: 세션 평균 음량
- `stability`: 전체 발성 시간 중 안전 구간 비율

`stability` 계산:

- `safeMs / voicedMs * 100`

즉 풍선 키우기는 최종 점수를 수치 하나로 환산하기보다, `얼마나 안정적으로 소리를 유지했는지`를 결과로 보여준다.

## 12. 현재 기획상 장점

- 텍스트 인식 없이 음량 조절만으로 바로 플레이 가능
- 발성을 오래 유지하는 훈련 경험이 직관적
- 본문 중심 UI라 사용자 집중도가 높음

## 13. 현재 구현상 주의점

### 1. 표시 dB는 절대값이 아님

게임 내부 보정값이므로 실제 물리 dB와 1:1 대응하지 않는다.

### 2. 성장 체감은 오디오 환경 영향을 받음

마이크 감도, 브라우저 자동 이득 조절, 주변 소음에 따라 체감이 달라질 수 있다.

### 3. 난이도 데이터의 shrinkPerSecond는 현재 거의 의미 없음

자동 감소를 제거한 상태라 실질적으로 사용되지 않는다.

## 14. 향후 개선 제안

- 안전 구간/위험 구간을 시각적으로 더 분명하게 표시
- 사용자별 마이크 민감도 보정
- 결과 리포트에 난이도별 등급 문구 추가

## 15. 한 줄 요약

현재 풍선 키우기는 `소리가 들어오면 즉시 성장`, `너무 크면 위험`, `시간 안에 목표 크기 도달` 구조를 가진 음량 조절형 발성 훈련 게임이다.
