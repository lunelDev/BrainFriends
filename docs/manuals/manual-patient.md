# 브레인프렌즈 환자 사용자 매뉴얼 v0.1

작성일: 2026-04-30
대상: 실어증·마비말장애 진단을 받은 분 (또는 보호자가 함께 보는 분)

## 1. 안내

브레인프렌즈는 의료진의 관리하에 언어재활 훈련을 도와주는 프로그램입니다. 진단이나 치료를 대신하지 않습니다.

## 2. 시작하기

### 2.1 로그인

1. 의료진이 안내한 아이디·비밀번호로 로그인합니다.
2. 카메라·마이크 권한을 허용해주세요.

### 2.2 첫 화면

- **오늘의 훈련**: 처방받은 훈련 step 으로 바로 진입
- **AAC 보드 (말 대신 심볼)**: 발화가 어려울 때 사용

## 3. 훈련 진행

### 3.1 음성 따라 말하기 (step-1, step-2)

1. 화면에 보이는 단어를 듣고 따라 말하기
2. 마이크에 가까이, 또렷하게 발음
3. 결과 표시까지 기다림

### 3.2 문장 산출 (step-4)

1. 그림이나 상황을 보고 한 문장으로 말하기
2. 30초 이내 발화 권장

### 3.3 AAC 보드 (말 대신 심볼)

1. **누구가 (주어)** + **무엇을 (의도)** + **어디서 (장소)** 순으로 심볼 선택
2. **commit** 버튼 → 의도 문장 자동 생성
3. 보호자가 이 문장을 함께 볼 수 있음 (옵션)

### 3.4 휴식·중단

> ⚠️ **중요 (RM-008 위험 통제)**: 피로 / 통증 / 불편 시 **즉시 중단** 하세요. 중단해도 결과는 부분 저장됩니다.

- 화면 우측 상단 **쉬기** 또는 **그만하기** 버튼
- 중단 후 이상반응 입력 화면 표시 → 솔직하게 입력 (의료진이 확인)

## 4. 자주 묻는 질문

| 질문 | 답 |
| --- | --- |
| 음성 인식이 자꾸 틀려요 | 마이크 거리 30cm + 주변 소음 줄이기. 그래도 틀리면 의료진이 검토합니다. |
| 카메라 화면이 안 떠요 | 권한 다시 허용 → 새로고침 |
| 결과가 안 보여요 | "재조회" 버튼 누르기 |
| 보호자가 보는 화면을 만들고 싶어요 | 의료진에게 보호자 링크 발급 요청 |
| 평가 받기 무서워요 | 의료진과 충분히 상담 후 시작하셔도 됩니다. |

## 5. 안내 사항

- 본 프로그램은 **진단 / 치료** 가 아니라 **훈련 보조** 입니다.
- AI 분석 결과는 의료진이 확인 후 판단합니다.
- 음성·영상은 훈련 useCase 의 경우 본인 기기에서만 처리되고 외부로 전송되지 않습니다.
- 주간 평가 (K-WAB) 의 음성은 안전한 보안 경로로 서버 분석됩니다.
- 본인의 데이터는 의료진과 본인 (보호자) 만 접근 가능합니다.
