# 브레인프렌즈 보호자 매뉴얼 v0.1

작성일: 2026-04-30
대상: 환자 가족 (보호자)

## 1. 안내

본인은 환자분의 보호자로서 브레인프렌즈 주간 리포트 링크를 받으셨습니다. 본 리포트는 **참고용**이며 의료적 판단을 위한 자료가 아닙니다.

## 2. 리포트 링크 사용

### 2.1 첫 접속

1. 의료진이 보내준 링크 (카카오톡 / 이메일) 클릭
2. **로그인 없이** 바로 read-only 페이지 진입
3. 링크는 7일 후 자동 만료
4. 링크 분실 시 의료진에게 재발급 요청

### 2.2 화면 구성

- **이번 주 훈련 횟수**
- **최근 K-WAB 점수 (AQ)** — 의료진 확정 후만 표시
- **AQ 변화 추이** — 4주 그래프
- **이상반응 보고** — 환자분이 입력한 불편 사항 (있을 경우)

## 3. 자주 묻는 질문

| 질문 | 답 |
| --- | --- |
| 점수가 떨어졌어요. 치료법을 바꿔야 하나요? | **아닙니다.** 본 리포트는 진단·치료 결정용이 아닙니다. 의료진과 상담해주세요. |
| 환자가 훈련을 안 해요 | 본 리포트는 격려·동기 부여용입니다. 강요하지 마세요. |
| 다른 가족과 공유해도 되나요? | 환자분 동의 후만 공유하시기 바랍니다. 링크는 토큰 기반이라 접근 이력이 기록됩니다. |
| 이상반응 보고가 있어요 | 의료진이 자동으로 알림을 받습니다. 별도 연락 필요 없습니다 (단, 응급 시 119). |
| 본인이 직접 입력할 수 있나요? | 보호자는 read-only 입니다. 입력은 환자 본인 또는 의료진만 가능합니다. |

## 4. 유의사항

> ⚠️ **중요 (RM-018 위험 통제)**: 본 리포트는 **치료 효과 변화**나 **처방 변경 근거**가 아닙니다. 의학적 판단은 반드시 의료진과 상담하시기 바랍니다.

- 환자분의 음성·영상은 보호자에게 노출되지 않습니다 (요약 점수만)
- 링크 URL을 공개 SNS / 단톡방에 올리지 마세요
- 링크 분실 시 즉시 의료진에게 알려 폐기 요청
