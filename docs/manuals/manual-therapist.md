# 브레인프렌즈 치료사·의사 사용자 매뉴얼 v0.1

작성일: 2026-04-30
대상: 언어치료사 (SLP), 재활의학과·신경과 의사

## 1. 제품 소개

브레인프렌즈는 뇌졸중·외상성 뇌손상 후 실어증·마비말장애 환자의 언어재활 훈련과 경과 모니터링을 보조하는 독립형 디지털의료기기소프트웨어 (개발 중) 입니다.

본 제품은 **임상 보조 도구**이며, 최종 진단 / 치료 결정 / 처방 변경은 본인 (의료진) 의 임상 판단에 따라 수행합니다.

## 2. 시작하기

### 2.1 로그인

1. 본인 자격으로 발급된 아이디·비밀번호로 로그인
2. 비밀번호 정책: 8자 이상 + 영문/숫자/특수문자 중 2종 이상 + 동일 문자 4회 반복 금지
3. 5회 연속 실패 시 15분 잠금
4. 30분 idle 시 자동 세션 만료

### 2.2 환자 검색·연결

1. **치료사 대시보드** → **환자 목록** → 환자 ID 또는 이름 검색
2. 본인 담당 환자만 표시됨 (admin 제외 — 비담당 치료사는 다른 환자 자료 조회 불가)
3. 신규 환자 연결: link-care 토큰 발급 → 환자 측 입력

## 3. 주요 기능

### 3.1 K-WAB 보조 점수 검토

> ⚠️ **중요 (RM-005 위험 통제)**: K-WAB 자동 점수는 보조 채점이며, 최종 진단은 본인의 임상 판단에 따릅니다. 자동 점수에 동의하지 않는 경우 **반려** 또는 **메모** 후 **재시행** 가능합니다.

**절차**:
1. 환자 상세 → 최근 주간 K-WAB 결과
2. 6 하위검사 + AQ/LQ/CQ 자동 산출 결과 검토
3. 측정 품질 표시 (`measured` / `partial` / `demo`) 확인 — `partial` 또는 `demo` 인 경우 임상가 검토 필수
4. 동의 시 **확정**, 의문 시 **메모 + 재평가 요청**

### 3.2 처방 발행

1. 환자 상세 → **처방 작성**
2. mode (집중훈련 / 일반훈련 / K-WAB) + 횟수 + 만료일
3. 발행 후 환자 측 즉시 표시
4. 처방 변경은 변경 이력 자동 기록 (`audit log`)

### 3.3 환자 history 조회

1. 환자 상세 → **history 탭**
2. 최근 7일 / 30일 / 전체 추이 그래프
3. 측정 품질 / 시선 응시율 / AAC 사용 빈도 / 이상반응 누적 표시
4. 부적절 데이터 (측정 실패 등) 식별 시 환자에게 재측정 안내

### 3.4 보호자 read-only 리포트 링크 발급

1. 환자 상세 → **보호자 링크 발급**
2. 토큰 기반 1회 링크 (만료 7일 / 수동 폐기 가능)
3. 보호자에게 카카오톡·이메일로 직접 전달
4. 발급·접근·폐기 이력은 audit log 자동 기록

### 3.5 이상반응 검토

1. 환자 상세 → **이상반응 탭**
2. 미해결 / 미확인 중증 건이 우선 표시
3. 의학적 판단 후 **확인 + 처치** 또는 **추가 평가 요청**
4. KAERS 보고 의무 사항은 별도 수동 보고 (severe 건은 본인이 직접 보고)

## 4. 측정 지표 해석 가이드

> ⚠️ **중요 (RM-013 위험 통제)**: 모든 보조 지표는 **참고용**이며 임상 맥락 함께 해석하시기 바랍니다.

| 지표 | 의미 | 임상 해석 시 주의점 |
| --- | --- | --- |
| 음성 transcript | WASM Whisper-tiny 또는 서버 Whisper 결과 | 발음장애·소음 시 오인식 가능. `reviewRequired=true` 표시 시 재평가 |
| 안면 측정 품질 | MediaPipe 478 랜드마크 신뢰도 | 조도·각도·얼굴 가림 시 저하. partial 이하는 환자 환경 점검 |
| 시선 응시율 (attention) | 화면 중앙 응시 비율 | irisDetected=false 비율 > 30% 시 환자 자세·조도 점검 |
| 반응시간 | step별 응답 latency | 기기/네트워크 영향 — 동일 환자 동일 환경 비교 권장 |
| AAC intent sentence | 규칙 기반 sequence → 한국어 문장 | AI 의도 예측 아님. unknownIds 누적 시 심볼 어휘 부족 |
| AQ (Aphasia Quotient) | K-WAB 자동 산출 | 본인 임상 평가와 ±10 이상 차이 시 정밀 평가 |

## 5. 사용 시 유의사항

- 본 제품은 **MCI / 치매 / 일반 인지개선** 목적 사용 금지
- 본 제품은 **응급 / 급성기 판단** 도구가 아님
- 본 제품 출력만으로 **자동 진단 / 자동 처방** 수행 불가
- 본인 환자 외 데이터 조회 시도는 audit log 기록 (관리자 검토 대상)
- 보호자 리포트 링크는 환자 동의 후 발급

## 6. 문제 해결

| 증상 | 조치 |
| --- | --- |
| 환자 결과 누락 | history 탭 → "재조회" 클릭 (서버 결과 우선 재조회) |
| K-WAB 점수가 임상 평가와 큰 차이 | 측정 품질 확인 → partial 이하면 재측정 안내 |
| WASM STT 모델 로딩 실패 | 환자 측 브라우저 새로고침 또는 server STT 모드 전환 (별도 안내) |
| 보호자 링크 만료 | 신규 발급 (이전 링크 자동 폐기) |
| 이상반응 보고 미수신 | 환자 측 알림 설정 + 본인 알림 설정 점검 |
