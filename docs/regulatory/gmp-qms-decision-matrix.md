# GMP / QMS 자체 vs 외주 결정 매트릭스 v0.2

작성일: 2026-04-30
최종 정정: 2026-05-07 (NIDS 상담노트 반영)
근거: 식약처 디지털의료기기 GMP 가이드라인 [별표3] / ISO 13485:2016 / 의료기기법 시행규칙 §15 / `NIDS_상담노트_브레인프렌즈.docx`
관련: `docs/regulatory/source-gmp-requirements.md`, `docs/regulatory/digital-medical-product-gap-matrix.md` §6 의사결정 3번, `docs/submission/gmp-outsourcing-rfp.md`
적용 범위: 1차 SaMD 신청 전 GMP 적합판정 획득 경로 결정

## 1. 목적

GMP/QMS 패키지 (품질매뉴얼, 8종 절차서, 형상관리, 변경관리, 공급업체 관리, 시정·예방조치, CAPA, 시판 후 감시) 를 자체 구축할지 외주 컨설팅으로 할지, 또는 하이브리드로 분리할지 PM 의사결정을 지원한다.

본 문서는 최종 결정서가 아니다. 비용·시간·시판 timing·인력 가용성을 종합해 사업 의사결정자가 확정한다.

원문 요구사항 단위 분해는 `source-gmp-requirements.md`를 기준으로 한다. 이 문서는 운영 방식의 선택지이고, 실제 개발·품질 gap은 원문 요구사항 매트릭스에서 관리한다.

## 1.1 2026-05-07 NIDS 상담노트 반영

NIDS 상담노트 기준으로 브레인프렌즈의 실행 순서는 다음처럼 잠근다.

| 순서 | 단계 | 의미 | PM 판단 |
| --- | --- | --- | --- |
| 1 | 제조업 허가 | 의료기기 제조업자로서의 회사 자격 | 즉시 준비. 품목 인허가보다 먼저 |
| 2 | GMP 적합인정 | 제조 및 품질관리 기준 | 1등급이 아닌 이상 필수로 보고 준비 |
| 3 | 품목 인허가 | 제품 자체의 인증/허가 | NIDS 영역. GMP 이후 신청 |

따라서 이 문서는 "품목 인허가 직전의 선택지"가 아니라, 제조업 허가 준비와 동시에 시작해야 하는 QMS 구축 의사결정 문서로 사용한다. NIDS 상담노트는 공식 서면 답변이 아니므로 외부 제출 근거가 아니라 내부 실행 기준으로만 활용한다.

## 2. 3 옵션 trade-off

| 항목 | A. 자체 구축 | B. 외주 컨설팅 (전부) | C. 하이브리드 (권장 1순위) |
| --- | --- | --- | --- |
| 초기 비용 | ₩2,000~3,500만 (GMP 컨설턴트 1명 PT 6개월) | ₩4,500~7,000만 (Big4 의료기기 컨설팅) | ₩2,500~4,000만 (코어 자체 + 심사 대응 외주) |
| 시판 timing | 6~9개월 | 4~6개월 (가장 빠름) | 5~7개월 |
| NIDS 상담노트 기준 실제 GMP 소요 | 한 번에 통과해도 약 3~4개월, 일반적으로 더 길어질 수 있음 | 일반 평균 6~9개월 | 5~7개월 기준으로 일정 산정 |
| 사후 변경 유연성 | 높음 (사내 절차서 직접 수정) | 낮음 (외주사 의존) | 중간 |
| 시판 후 운영 비용 | 낮음 (사내 인력) | 높음 (월 ₩300~600만 retainer) | 중간 |
| 식약처 GMP 심사 통과율 | 첫 신청 60~70% | 첫 신청 90%+ | 첫 신청 80%+ |
| 인허가 컨설턴트 협상력 | 약함 | 강함 (외주사가 협상 대행) | 중간 |
| 1차 SaMD 신청 적합 | 가능 | 가능 | 가능 |
| DTx 분류 추가 신청 적합 | 어려움 (전향적 임상 + GMP 심화 필요) | 적합 (외주사가 전 영역 처리) | 가능 (외주가 임상·심사 부분만) |
| 형상관리 시스템 구축 | 자체 (release manifest 이미 결정성 함수 보유 ✅) | 외주사 표준 도구 (보통 별도 라이선스) | 자체 release manifest + 외주 검증 |
| ISO 13485 인증 별도 진행 | 별도 패스 가능 (선택) | 외주 패키지 포함 | 외주 부분 포함 |

## 4. 의사결정 변수

다음 5개 변수의 가중치에 따라 최종 옵션이 갈린다.

| 변수 | 영향 | 측정 지표 |
| --- | --- | --- |
| 시판 timing 우선순위 | B > C > A | 매출 시작 시점 D-day |
| 예산 가용성 | A < C < B | 1년 누적 비용 |

| 사후 변경 빈도 예측 | A > C > B | 분기당 변경 건수 예측 |

## 5. PM 권고

### 5.1 1차 권고: **C 하이브리드** (다음 조건 충족 시)

- 사내 PM 1.0 FTE + QM/SE 0.5 FTE 가능 (현재 골든브레인케어 인력 기준)
- 1차 SaMD 단독 신청 (DTx 첨부자료는 후속) 결정
- 결정성 V&V / release manifest / CVE 면제 / 위험관리 파일 / IRT / 사용성평가 등 코드 산출물 보유 → 외주사 의존도 낮춤
- NIDS 상담노트 기준으로 제조업 허가와 GMP가 품목 인허가보다 먼저 필요하므로, 내부 기술문서는 자체가 만들고 품질매뉴얼·심사 대응은 외주 검토를 받는 방식이 비용 대비 효율적이다.

### 5.2 2차 권고: **B 외주** (다음 조건 시)

- 시판 timing 이 6개월 내 절대 목표
- 예산 ₩7,000만+ 확보
- DTx 분류 동시 신청 (전향적 임상 + GMP 심화)
- 사내 GMP 인력 부재

### 5.3 비권고: **A 자체** (다음 조건만 가능)

- 식약처 사전상담 + IRB 경험 보유 PM 사내 존재
- 시판 timing 9개월+ 여유
- 첫 신청 보완 (60~70% 통과율) 감수

## 6. 외주사 후보 (B / C 옵션 시)

본 §6 은 PM 사전 조사 결과로 갱신. 현재 후보 매트릭스는 별도 비공개 파일 (`gmp-outsourcing-rfp.md` v0.1) 참조.

## 8. 갱신 트리거

- 외주사 견적 수령 → §5 가중치 재산정
- 사내 인력 변동 → §4 변수 재평가
- DTx 분류 동시 신청 결정 → §5.2 자동 활성화
- 식약처 사전상담 회신에서 GMP 요구사항 변경 → §3 산출물 표 갱신
