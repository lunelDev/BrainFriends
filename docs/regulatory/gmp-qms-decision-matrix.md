# GMP / QMS 자체 vs 외주 결정 매트릭스 v0.1

작성일: 2026-04-30
근거: 식약처 디지털의료기기 GMP 가이드라인 [별표3] / ISO 13485:2016 / 의료기기법 시행규칙 §15
관련: `docs/regulatory/digital-medical-product-gap-matrix.md` §6 의사결정 3번, `docs/submission/gmp-outsourcing-rfp.md`
적용 범위: 1차 SaMD 신청 전 GMP 적합판정 획득 경로 결정

## 1. 목적

GMP/QMS 패키지 (품질매뉴얼, 8종 절차서, 형상관리, 변경관리, 공급업체 관리, 시정·예방조치, CAPA, 시판 후 감시) 를 자체 구축할지 외주 컨설팅으로 할지, 또는 하이브리드로 분리할지 PM 의사결정을 지원한다.

본 문서는 최종 결정서가 아니다. 비용·시간·시판 timing·인력 가용성을 종합해 사업 의사결정자가 확정한다.

## 2. 3 옵션 trade-off

| 항목 | A. 자체 구축 | B. 외주 컨설팅 (전부) | C. 하이브리드 (권장 1순위) |
| --- | --- | --- | --- |
| 초기 비용 | ₩2,000~3,500만 (GMP 컨설턴트 1명 PT 6개월) | ₩4,500~7,000만 (Big4 의료기기 컨설팅) | ₩2,500~4,000만 (코어 자체 + 심사 대응 외주) |
| 시판 timing | 6~9개월 | 4~6개월 (가장 빠름) | 5~7개월 |
| 사후 변경 유연성 | 높음 (사내 절차서 직접 수정) | 낮음 (외주사 의존) | 중간 |
| 시판 후 운영 비용 | 낮음 (사내 인력) | 높음 (월 ₩300~600만 retainer) | 중간 |
| 식약처 GMP 심사 통과율 | 첫 신청 60~70% | 첫 신청 90%+ | 첫 신청 80%+ |
| 인허가 컨설턴트 협상력 | 약함 | 강함 (외주사가 협상 대행) | 중간 |
| 1차 SaMD 신청 적합 | 가능 | 가능 | 가능 |
| DTx 분류 추가 신청 적합 | 어려움 (전향적 임상 + GMP 심화 필요) | 적합 (외주사가 전 영역 처리) | 가능 (외주가 임상·심사 부분만) |
| 형상관리 시스템 구축 | 자체 (release manifest 이미 결정성 함수 보유 ✅) | 외주사 표준 도구 (보통 별도 라이선스) | 자체 release manifest + 외주 검증 |
| ISO 13485 인증 별도 진행 | 별도 패스 가능 (선택) | 외주 패키지 포함 | 외주 부분 포함 |

### 3.2 외주 컨설팅 (B) 시 분담

전 항목 외주사 작성 + 우리는 (1) 코드 산출물 제공, (2) 검토·서명, (3) 심사 인터뷰 응답.

### 3.3 하이브리드 (C) 시 분담 — **권장**

| 외주 부분 | 자체 부분 |
| --- | --- |
| 품질매뉴얼 (가장 표준적) | 설계 및 개발 절차서 (코드 산출물과 직결) |
| GMP 심사 인터뷰 대응 | 형상관리 / 보안지침 / SOUP / CVE 면제 (이미 ✅) |
| ISO 13485 인증 신청 (선택) | 변경관리 계획서 (이미 결정성 함수 ✅) |

| 식약처 사전상담 동행 | 위험관리 파일 (이미 v1.0 진행 중) |

## 4. 의사결정 변수

다음 5개 변수의 가중치에 따라 최종 옵션이 갈린다.

| 변수 | 영향 | 측정 지표 |
| --- | --- | --- |
| 시판 timing 우선순위 | B > C > A | 매출 시작 시점 D-day |
| 예산 가용성 | A < C < B | 1년 누적 비용 |
| 사내 인력 (PM/QM/SE) | A 가능 ≥ 1.5 FTE / C 가능 ≥ 0.8 FTE / B 0 FTE | 인력 가용 FTE |
| DTx 분류 동시 신청 | B 강력 권장 | 임상 자료 강도 |
| 사후 변경 빈도 예측 | A > C > B | 분기당 변경 건수 예측 |

## 5. PM 권고

### 5.1 1차 권고: **C 하이브리드** (다음 조건 충족 시)

- 사내 PM 1.0 FTE + QM/SE 0.5 FTE 가능 (현재 골든브레인케어 인력 기준)
- 1차 SaMD 단독 신청 (DTx 첨부자료는 후속) 결정
- 결정성 V&V / release manifest / CVE 면제 / 위험관리 파일 / IRT / 사용성평가 등 코드 산출물 보유 → 외주사 의존도 낮춤

### 5.2 2차 권고: **B 외주** (다음 조건 시)

- 시판 timing 이 6개월 내 절대 목표
- 예산 ₩7,000만+ 확보
- DTx 분류 동시 신청 (전향적 임상 + GMP 심화)
- 사내 GMP 인력 부재

### 5.3 비권고: **A 자체** (다음 조건만 가능)

- 식약처 사전상담 + IRB 경험 보유 PM 사내 존재
- 시판 timing 9개월+ 여유
- 첫 신청 보완 (60~70% 통과율) 감수

## 6. 외주사 후보 (B / C 옵션 시)
