# 브레인프렌즈 사용목적·대상환자·금기 및 주의사항 초안

작성일: 2026-04-29  
상위 문서: `docs/regulatory/claim-lock.md`, `docs/regulatory/digital-medical-product-gap-matrix.md`  
문서 성격: 허가·심사, IRB, 임상시험계획서, 제품설명서 작성을 위한 내부 기준 초안

## 1. 목적

이 문서는 브레인프렌즈의 1차 허가 범위를 좁히고, 사용목적·대상 환자·사용자·사용 환경·금기 및 주의사항을 일관되게 정의하기 위한 내부 기준 문서이다.

현재 PM 판단은 다음과 같다.

- 1차 범위는 뇌졸중, 외상성 뇌손상, 기타 뇌질환 이후 발생한 실어증·마비말장애 재활 보조로 제한한다.
- MCI, 치매 예방, 일반 인지개선, VR/AR 기반 치료는 1차 허가 범위에서 제외한다.
- 제품은 의료진의 평가와 치료 결정을 보조하는 소프트웨어이며, 자동 진단·자동 처방·의료진 대체 제품으로 정의하지 않는다.

최종 사용목적 문구는 식약처 품목분류, 사전상담, 임상시험계획 검토 결과에 따라 조정한다.

## 2. 제품 개요

브레인프렌즈는 환자용 훈련 화면, 치료사·의사용 대시보드, 보호자용 read-only 리포트 링크로 구성된 독립형 디지털의료기기소프트웨어이다.

제품은 음성, 안면 움직임, 반응시간, 시선, AAC 심볼 입력을 수집하여 언어재활 훈련 수행 상태와 경과 확인을 보조한다. 분석 결과는 치료사 또는 의사의 임상적 판단을 돕기 위한 참고 지표이며, 최종 평가와 치료 결정은 의료진이 수행한다.

## 3. 1차 사용목적 잠금 문구

### 3.1 내부 기준 문구

브레인프렌즈는 뇌졸중, 외상성 뇌손상, 기타 뇌질환 이후 발생한 실어증 또는 마비말장애 환자를 대상으로, 의료진의 관리하에 언어재활 훈련 수행과 경과 모니터링을 보조하기 위해 사용하는 독립형 디지털의료기기소프트웨어이다.

본 제품은 음성, 안면 움직임, 반응시간, 시선, AAC 심볼 입력 및 K-WAB 기반 과제 수행 정보를 활용하여 재활 훈련 지표와 보조 채점 결과를 제공한다. 제품이 제공하는 정보는 의료진의 임상적 판단을 보조하기 위한 참고 자료이며, 질환의 확정 진단, 치료 처방, 응급 판단을 대체하지 않는다.

### 3.2 대외 설명용 축약 문구

브레인프렌즈는 뇌질환 이후 발생한 실어증·마비말장애 환자의 언어재활 훈련과 경과 모니터링을 의료진 관리하에 보조하는 독립형 디지털의료기기소프트웨어로 개발 중입니다.

## 4. 대상 환자

### 4.1 포함 대상

| 항목 | 기준 |
| --- | --- |
| 질환 또는 상태 | 뇌졸중, 외상성 뇌손상, 기타 뇌질환 이후 발생한 실어증 또는 마비말장애 |
| 재활 단계 | 급성 응급 처치 이후 안정화되어 언어재활 훈련이 가능한 상태 |
| 사용 능력 | 화면 안내를 이해하거나 보호자·치료사 보조하에 과제를 수행할 수 있는 사용자 |
| 입력 능력 | 음성, 터치, 시선 응시, AAC 심볼 선택 중 하나 이상의 입력을 사용할 수 있는 사용자 |
| 관리 조건 | 치료사, 의사, 또는 지정된 재활 담당자의 평가·관리 계획에 포함된 사용자 |

### 4.2 후속 적응증 후보

다음 대상은 현재 1차 허가 범위에서 제외하고, 별도 임상·성능 근거가 확보된 뒤 후속 적응증으로 검토한다.

- 경도인지장애(MCI)
- 치매 예방 또는 인지기능 개선 목적 사용자
- 일반 고령자 인지훈련
- 소아 언어발달장애
- 파킨슨병, ALS 등 진행성 신경질환의 발화 기능 저하
- VR/AR/HMD 기반 재활 치료 대상자

| 관리자 | 계정, 권한, 보안, 시스템 상태 관리 | 사용자 관리, 감사로그, 보안 설정, 장애 대응 |

| 주변 환경 | 얼굴과 입 주변이 카메라에 보이고, 과제 음성이 녹음 가능한 조도·소음 수준 |

## 7. 출력 정보

브레인프렌즈가 제공하는 출력은 의료진 참고용 보조 지표이다.

| 출력 | 설명 | 사용 경계 |
| --- | --- | --- |
| 음성 전사 | 과제 발화의 STT 기반 전사 결과 | 전사 오류 가능성이 있으므로 치료사 검토 필요 |
| 조음·음향 지표 | 발화 정확도, 음향 특성, 과제 수행 관련 보조 지표 | 단독 진단 지표로 사용 금지 |
| 안면 움직임 지표 | 입술 개폐, 구강 움직임, 조음 관련 얼굴 움직임 | 카메라 위치·조도에 영향 가능 |
| 시선 지표 | 화면 중앙 응시 비율, off-task 시간, 측정 품질 | 주의집중 참고 지표이며 인지기능 진단 지표 아님 |
| 반응시간 | 과제 제시 후 사용자 반응까지의 시간 | 피로, 기기 성능, 네트워크 영향 가능 |
| AAC 문장 | 심볼 선택 sequence 기반 규칙형 의도 문장 | AI 예측이 아니라 사용자 선택 기반 변환 |
| K-WAB 보조 점수 | K-WAB 기반 과제 수행 결과의 자동 계산값 | 최종 평가는 의료진 검토 후 확정 |
| 경과 리포트 | 세션 수행률, 점수 변화, 이상반응 여부 요약 | 치료 계획 보조용이며 치료 효과 입증 자료로 단독 사용 금지 |

## 8. 금기 및 사용 제한

### 8.1 사용 금기

다음 사용자는 제품 사용 대상에서 제외하거나 의료진의 별도 판단이 필요하다.

| 항목 | 사유 |
| --- | --- |
| 응급 또는 급성기 의학적 판단이 필요한 환자 | 본 제품은 응급 판단 도구가 아니다. |
| 의식 저하, 섬망, 급성 혼돈 등으로 과제 수행이 불가능한 환자 | 과제 결과의 신뢰성을 확보하기 어렵다. |
| 중증 청각장애로 음성 과제 지시를 이해할 수 없는 경우 | 음성 기반 과제 수행이 제한된다. |
| 중증 시각장애로 화면 자극을 확인할 수 없는 경우 | 화면 기반 과제와 시선 지표가 제한된다. |
| 카메라 사용이 불가능하거나 얼굴 촬영에 동의하지 않는 경우 | 안면·시선 지표 산출이 불가능하다. |
| 마이크 사용이 불가능하거나 음성 녹음에 동의하지 않는 경우 | 음성 분석과 STT 기반 지표 산출이 불가능하다. |
| 심한 불안, 피로, 통증 등으로 훈련 지속이 어려운 경우 | 훈련 중단 및 의료진 확인이 필요하다. |

### 8.2 사용 제한

다음 조건에서는 결과 해석에 주의한다.

- 주변 소음이 큰 환경
- 얼굴 일부가 가려진 상태
- 카메라 각도 또는 조도가 불안정한 상태
- 마이크 입력 품질이 낮은 상태
- 인터넷 연결이 불안정한 상태
- 보호자 또는 치료사가 정답을 유도한 상태
- 환자가 과제를 이해하지 못했거나 피로를 호소한 상태
- STT 결과와 실제 발화가 다르게 보이는 상태

## 9. 주의사항

| 주의사항 | 설명 |
| --- | --- |
| 의료진 검토 필요 | 제품 출력은 보조 정보이며 최종 평가와 치료 결정은 의료진이 수행한다. |
| 진단 목적 사용 금지 | 본 제품 결과만으로 실어증, 마비말장애, 인지장애를 확정 진단하지 않는다. |
| 치료 효과 단독 판단 금지 | 점수 변화는 참고 지표이며 치료 효과 입증은 임상시험 설계에 따라 판단한다. |
| STT 오류 가능성 | 음성 전사는 발음, 소음, 마이크 품질, 방언, 말속도에 따라 오류가 발생할 수 있다. |
| 안면·시선 지표 한계 | 조도, 카메라 위치, 안경, 얼굴 가림 등에 따라 측정 품질이 저하될 수 있다. |
| 개인정보 보호 | 음성, 영상, 훈련 결과는 민감한 건강정보로 관리해야 한다. |
| 이상반응 보고 | 피로, 어지러움, 불안, 통증, 사용 중 불편이 발생하면 사용을 중단하고 의료진에게 알린다. |
| 보호자 열람 제한 | 보호자 리포트는 read-only 정보 제공용이며 치료 판단 또는 처방 변경 권한을 부여하지 않는다. |

## 10. 의료진 개입 지점

브레인프렌즈는 다음 지점에서 의료진 또는 재활 담당자의 검토가 필요하다.

| 시점 | 의료진 개입 |
| --- | --- |
| 최초 사용 전 | 대상 환자 적합성, 사용 환경, 금기·주의사항 확인 |
| 훈련 계획 설정 | 과제 종류, 난이도, 빈도, 기간 지정 |
| K-WAB 또는 평가 과제 후 | 자동 계산 결과와 실제 수행 내용 검토 |
| 급격한 점수 변화 발생 | 측정 오류, 건강 상태 변화, 과제 이해 실패 여부 확인 |
| 이상반응 또는 사용 불편 보고 | 사용 중단 여부와 추가 조치 판단 |
| 보호자 문의 발생 | 경과 정보 해석과 치료 계획 변경 여부 판단 |
| STT 또는 영상 분석 실패 반복 | 대체 입력, AAC 사용, 오프라인 평가 필요 여부 판단 |

| 평가 결과 | 경과 리포트와 치료사 검토 | 최종 임상 판단은 의료진이 수행한다. |
| 보호자 리포트 | 경과 확인 | read-only 링크 접근으로 제한한다. |

## 12. IRB/임상 문서용 범위 문구

본 연구 또는 사용성 평가는 뇌질환 이후 실어증 또는 마비말장애가 있는 사용자를 대상으로, 브레인프렌즈 소프트웨어의 언어재활 훈련 보조 기능, 사용성, 안전성, 수행 지표 변화를 탐색적으로 평가한다.

브레인프렌즈가 제공하는 음성, 안면, 반응시간, 시선, AAC, K-WAB 기반 결과는 의료진 검토를 위한 참고 정보이며, 연구 참여자의 질환 진단, 치료 처방, 응급 판단을 대체하지 않는다.

## 13. 제외 문구

다음 문구는 본 사용목적 문서와 충돌하므로 허가·IRB·대외 자료에서 사용하지 않는다.

- 실어증 자동 진단 소프트웨어
- 마비말장애 자동 진단 소프트웨어
- 의료진 없이 사용하는 치료 소프트웨어
- 치료 효과가 입증된 디지털치료기기
- MCI 또는 치매 예방 치료 제품
- 전 제품 원본 데이터 미전송 온디바이스 AI
- AI가 치료 방향을 자동 결정
- 보호자가 치료 결과를 판정

우선 포함할 위험 항목은 다음과 같다.

- STT 전사 오류로 인한 잘못된 평가
- 안면·시선 측정 실패로 인한 결과 왜곡
- 환자 피로 또는 불편 누락
- 보호자 링크 오남용
- 권한 없는 의료정보 접근
- AI 보조 지표를 진단 결과로 오해
- 네트워크 또는 저장 실패로 인한 기록 누락
- 부적절한 대상자 사용
