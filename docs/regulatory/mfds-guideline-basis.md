## 3. 브레인프렌즈 고정 판단

| 항목 | 고정 기준 |
| --- | --- |
| 제품 포지션 | 독립형 디지털의료기기소프트웨어(SaMD) |
| DTx 해석 | 별도 트랙이 아니라 SaMD 안의 치료 목적 제품 성격 |
| 예상 등급 | 2등급 가능성을 기준으로 준비하되 개정안 기준 자체 선언 후 심사에서 확정 |
| 사용목적 | 의료진 관리하 언어재활 훈련 수행 및 경과 모니터링 보조 |
| STT 정책 | 현재 제품 기본값은 서버 보안 프록시 STT |
| AI 역할 | 진단/처방/치료 결정이 아니라 치료사 검토용 보조 지표 |
| 임상 | 치료/재활 효과를 표방하면 사실상 필요 |
| QMS/GMP | 제조업 허가 → GMP → 품목 인허가 순차 진행 |

## 4. 문서 작성 규칙

- `permit-readiness-internal-standard`, `gap-matrix`, `claim-lock`, `gmp-qms-decision-matrix`는 이 문서를 기준으로만 갱신한다.
- 새 문서에서 WASM-STT, 완전 온디바이스 STT, 자동 진단, 허가 완료, 치료효과 입증 표현을 기본값처럼 쓰지 않는다.
- 분류·등급 판단은 `개정안(최종)`을 우선 보고, 현행본은 비교용으로만 참조한다.
- AI 또는 ML 기능이 포함된 제출 문서에는 최소한 다음 6개 항목을 별도 표 또는 섹션으로 명시한다.
  1. 임상적 관점에서의 성능
  2. 작용원리에 기반한 세부 프로그램의 주요 기능
  3. 해당 제품에 적용되는 디지털 기술의 세부적인 특징
  4. 학습데이터의 정보
  5. 학습데이터의 업데이트 예상 주기
  6. 클라우드 서비스 종류
