## 2. 사용성평가 개요 (IRB 요약)

- 평가명: 브레인프렌즈 (실어증·마비말장애 재활 보조 SaMD) 사용적합성 평가
- 평가 단계: formative (반복 5/5/5~8) + summative (확정 15명, U1 환자 8 + U2 치료사 4 + U3 보호자 3)
- 모집 대상: U1 60~80대 5 + 50대 이하 3 (실어증/마비말장애 진단), U2 언어치료사 또는 재활의학과·신경과 의사, U3 환자 가족 (50~70대)
- 평가 환경: 사용성 랩 또는 외래 클리닉 시뮬레이션, 1회 60~90분
- 데이터 수집: think-aloud, 화면 녹화 (음성 포함), 시선 (선택), 후속 인터뷰 + SUS 설문
- 보관 기간: 평가 종료 + 보고서 확정 + 6개월 → 폐기
- 위해성 등급: 최소위험 (의료기기 사용 보조, 침습적 요소 없음)

## 3. 피험자 동의서 양식 (별첨 A)

### 3.1 동의 항목 체크리스트

```
[ ] 평가 목적과 절차 설명을 들었음
[ ] 평가 시간 (60~90분) 과 회차 정보 확인
[ ] 화면·음성 녹화에 동의 (보관 기간: 평가 종료 후 6개월)
[ ] 시선 추적 사용에 동의 (선택)
[ ] 본인 식별 정보 (이름, 전화번호) 가 가명 ID 로 분리 저장됨을 이해
[ ] 평가 결과가 식약처 인허가 자료로 사용될 수 있음을 이해
[ ] 언제든지 사유 설명 없이 중단 가능함을 이해
[ ] 동의 철회 시 데이터 즉시 폐기 절차 안내 받음
[ ] 보상 (있을 경우 §7) 정책 안내 받음
[ ] 본 평가가 의료적 진단·치료 목적이 아님을 명확히 인식
```

### 3.2 서명 양식

```
참여자 성명: ______________  서명: ______  날짜: ______
법정대리인 (필요 시): ______  서명: ______  날짜: ______
평가자 성명: ______________  서명: ______  날짜: ______
```

### 3.3 권리 안내문 (한국어)

> 본인은 본 사용성평가에 자발적으로 참여하며, 다음 권리를 보유함을 인지합니다.
> (1) 언제든지 사유 설명 없이 평가를 중단할 권리
> (2) 동의 철회 시 본인 관련 데이터의 즉시 폐기 요청 권리
> (3) 평가 진행 중 또는 종료 후 본인 데이터의 사용 범위와 보관 기간에 관한 정보 제공 요청 권리
> (4) 본 평가가 의료기기로 허가되지 않은 개발 단계 소프트웨어임을 이해하며, 본 평가 결과로 본인의 의학적 상태에 대한 진단 또는 치료 결정이 이루어지지 않음을 인지

## 5. 데이터 처리 명세

### 5.1 수집 데이터

| 데이터 | 식별성 | 처리 방법 |
| --- | --- | --- |
| 가명 ID (P001~P015 등) | 비식별 | 평가 데이터셋에 사용 |
| 이름·전화번호·생년월 | 식별 | 별도 lookup 테이블 (오프라인 보관, 연구원 1인 접근) |
| 화면 녹화 (영상+음성) | 준식별 | 가명 ID 부여, 얼굴/음성 비식별화는 분석 후 폐기 |
| 시선 추적 데이터 | 비식별 | 가명 ID 만 |
| Think-aloud 메모 | 비식별 | 평가자 손글씨 → OCR → 가명 ID 만 |
| SUS 설문 결과 | 비식별 | 가명 ID 만 |
| Use error 기록 | 비식별 | `useScenarioValidator.ts` UseErrorRecord 양식 |

### 5.2 PHI 마스킹 적용

`src/lib/server/phiMasking.ts` 의 `maskPhiObject` 가 `report` 출력 시 자동 적용 — 이름/전화/RRN/이메일/환자ID 식별자 마스킹 (SR-PHI-013).

## 6. 위해성 평가

### 6.1 평가 활동 자체 위해

| 위해 | 가능성 | 보호 조치 |
| --- | --- | --- |
| 환자 피로·불편 (60~90분 평가) | 중 | 30분 단위 휴식 권장, 즉시 중단 가능 |
| 평가 중 심리적 스트레스 (난해한 작업) | 중 | 평가자 격려, 작업 실패가 본인 잘못이 아님 명시 |
| 보호자 대신 답변 (U1 환자 평가 시 영향) | 낮 | 평가자가 동석 보호자에게 사전 안내 |
| 화면 녹화 사생활 노출 우려 | 낮 | 동의서에 보관 기간 + 폐기 절차 명시 |

### 6.2 RM-* 매핑

본 평가는 다음 위해요인 통제를 검증한다 (`risk-management-file.md` §11.7 매핑):
- RM-005 K-WAB 진단 오해 (POF-07 critical)
- RM-006 AAC 문장 생성 오해 (POF-04 primary)
- RM-008 환자 피로 (POF-05 critical, POF-06 primary)
- RM-013 AI 보조 지표 과신 (POF-07/POF-12)
- RM-018 부정확한 보호자 해석 (POF-12 primary)

## 8. 데이터 보관·폐기 정책

- 평가 종료 + 보고서 확정 (D 일) → D + 6개월 시점 자동 폐기
- 폐기 절차: (1) 영상/음성 파일 sha256 검증 후 secure delete, (2) lookup 테이블 종이 파쇄, (3) 가명 ID 평가 데이터셋은 비식별 산출물로 영구 보관
- 동의 철회 시 즉시 폐기 (D + 0)
- 폐기 증적: `audit-log` (auditChain.ts) 에 폐기 시각 + 평가자 + 가명 ID 목록 기록 (`SR-SEC-UC07`)

## 9. 평가자 자격 / 훈련

- 평가자 학력: 인지과학·HCI·임상심리 석사 이상 권장
- IRB 윤리 교육 이수 (CITI 또는 KAIRB 과정)
- 본 프로토콜 사전 학습 (`usability-evaluation-protocol.md` 12 시나리오)
- think-aloud 진행 훈련 (시나리오 별 안내문, 힌트 금지 원칙)
- Use error 분류 훈련 (severity minor/moderate/severe + hazardLinks RM-* 매핑)
