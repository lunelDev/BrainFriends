# Source Post-Stroke Dysarthria DTx Requirements Matrix

작성일: 2026-05-08  
원문: `뇌졸중+후+마비말장애+개선+디지털치료기기+안전성·성능+평가+및+임상시험계획서+작성+가이드라인.pdf`  
목적: 뇌졸중 후 마비말장애 개선 디지털치료기기 트랙에 들어갈 때 필요한 안전성·성능 평가와 임상시험계획서 요구사항을 브레인프렌즈 개발 항목으로 분해한다.

## 1. 해석 원칙

- 이 문서는 브레인프렌즈 전체를 즉시 DTx로 확정하는 근거가 아니다.
- 직접 적용 범위는 `뇌졸중 후 마비말장애 개선` 클레임이다. 실어증, 일반 언어재활, 노래훈련 전체 효과로 과확장하지 않는다.
- 원문 예시는 2025년 12월 기준 민원인 안내서이며, 실제 대상자 수·평가변수·시험설계는 제품 특성, 탐색임상, 식약처 사전상담 결과로 확정해야 한다.
- 현재 대외 표현은 “재활 보조 SaMD”로 유지하고, DTx/치료효과 표현은 임상시험 근거 확보 전까지 조건부 또는 금지로 둔다.

## 2. 제품 적용 결론

브레인프렌즈가 마비말장애 DTx 트랙을 노리려면 개발 우선순위는 다음처럼 바뀐다.

1. 말 명료도, MPT, DDK, PCC를 수집·저장·내보내기 가능한 구조로 만든다.
2. 각 훈련 세션의 사용량, 사용횟수, 수행시간, 중단사유, 순응도를 자동 기록한다.
3. 부작용/이상사례, 피로, 좌절감, 안구·목·턱 피로 등 안전성 이벤트를 수집한다.
4. 치료사 또는 평가자가 원음, target, transcript, 채점근거를 검토할 수 있어야 한다.
5. 확증임상 전에는 내부 전후 비교 자료와 탐색적 사용성/성능 자료를 먼저 만든다.
