# UI 설계 보완 요약

## 목적

이 문서는 현재 BrainFriends 프로젝트에서 UI/UX 측면으로 아직 부족한 부분과, 있으면 좋은 보완 항목을 짧게 정리한 메모입니다.

공인성적서 대응 구조와 별개로, 실제 서비스 완성도와 사용성 측면에서 어떤 화면을 더 손봐야 하는지 빠르게 확인하기 위한 용도입니다.

## 1. 현재 상태 한 줄 요약

- 치료사 화면은 `운영/해석용 구조`가 꽤 갖춰졌으나, 화면 간 상태 연동과 실제 액션 기능은 더 보완할 여지가 있습니다.
- 사용자 화면은 `홈/결과` 위주로 일부 정리됐지만, 인포그래픽 수준의 완성형 UX까지는 아직 아닙니다.
- step 훈련 화면은 의도적으로 기존 집중형 디자인을 유지하고 있습니다.

## 2. 치료사 UI에서 부족한 부분

### 우선 보완이 필요한 항목

- 대시보드에서 선택한 검색 조건과 필터가 `사용자 목록`, `결과 화면`까지 더 자연스럽게 이어지는 흐름
- `저장 실패 재시도`, `결과 다운로드` 외에 실제 운영 액션 확장
  - 리포트 다운로드
  - 후속 과제/조치 배정
  - 특정 사용자 기준 빠른 이동
- 메인 대시보드와 결과 화면의 시각 계층 통일
- 결과 화면의 필터 조합 상태를 더 잘 보여주는 요약 영역

### 있으면 좋은 항목

- 오늘 확인이 필요한 사용자만 따로 모아보는 `집중 확인 리스트`
- 최근 오류/저장 실패를 우선순위로 보여주는 운영 카드
- 사용자 상세에서 `최근 세션 비교`를 더 빠르게 읽게 하는 시각 정리
- 대시보드에서 사용자 상세 패널을 한 명 이상 비교하는 기능

## 3. 사용자 UI에서 부족한 부분

### 우선 보완이 필요한 항목

- 홈 화면 카드 간격/정보량을 더 단순하게 정리
- 결과 리포트 3종의 톤과 구성 통일
  - 자가진단
  - 언어 재활
  - 브레인 노래방
- `마이페이지 / 내 재활 관리` 화면 부재
- 홈에서 추천 훈련과 최근 진행 내역의 우선순위 정리

### 있으면 좋은 항목

- 연속 훈련, 목표 달성, 최근 성장 변화 같은 동기부여 요소 강화
- 최근 결과에서 바로 이어서 훈련하는 빠른 동선
- 목표 설정/훈련 추천을 더 명확하게 보여주는 카드
- 측정 품질이 낮을 때 사용자에게 자연스럽게 설명하는 보조 문구

## 4. step 훈련 화면 관련 판단

- step 화면은 현재처럼 `집중형 인터랙션`을 유지하는 판단이 맞습니다.
- 과도한 상단 요약 카드나 대시보드형 UI를 다시 넣을 필요는 없습니다.
- 다만 아래 항목 정도는 필요하면 보완할 수 있습니다.
  - 진행률 표시의 일관성
  - 버튼 크기/간격 정리
  - 오류/재시도 메시지 톤 통일

## 5. 내일 바로 보면 좋은 우선순위

1. 사용자 결과 화면 3종 톤 통일
2. 사용자 홈 화면 정보 밀도 정리
3. 치료사 결과 화면 필터/요약 상태 고도화
4. 치료사 액션 기능 범위 확대 여부 결정
5. 마이페이지/내 재활 관리 화면 필요 범위 확정

## 6. 한 줄 결론

현재 프로젝트는 `공인성적서 대응 구조`가 UI보다 앞서 있고, UI는 기본 뼈대는 있지만 사용자 경험 완성도와 운영 편의성 측면에서 아직 더 다듬을 부분이 남아 있습니다.
