# 시험기관 선정 기준표

## 목적

이 문서는 BrainFriends의 공인성적서 및 SaMD 준비를 위해, 시험기관이 아직 정해지지 않은 상태에서 어떤 기준으로 기관을 비교해야 하는지 정리한 기준표입니다.

## 우선 확인 항목

### 1. 시험 가능 범위

아래 항목을 모두 다룰 수 있는지 먼저 확인합니다.

- SW V&V
- 사이버보안
- AI 성능평가

한 기관에서 모두 가능하면 가장 효율적이고, 아니면 항목별 분리 진행이 가능한지 확인합니다.

### 2. 디지털의료기기 / SaMD 경험

- 디지털의료기기소프트웨어 또는 SaMD 관련 경험이 있는지
- 소프트웨어 중심 제품의 시험/성적서 발급 경험이 있는지
- AI 기능 포함 제품 검토 경험이 있는지

### 3. 사전 상담 가능 여부

- 시험 전에 제품 개요와 사용목적 문구를 기준으로 상담이 가능한지
- 필요한 성적서 종류를 먼저 가이드해 줄 수 있는지
- 시험 범위와 문서 형식을 사전 확인해 줄 수 있는지

### 4. 제출 문서 형식

- 기관이 원하는 문서 양식이 있는지
- 결과서/부록/증적 포맷이 어떻게 되는지
- 사전 문의 단계에서 어떤 문서를 먼저 보내면 되는지

### 5. 일정 / 비용 / 소통

- 예상 일정
- 예상 비용
- 담당자 응답 속도
- 추가 질의응답 가능 여부

## 현재 추천 판단 기준

1. 세 항목(SW V&V / 사이버보안 / AI 성능평가)을 모두 다룰 수 있는지
2. 디지털의료기기 / SaMD 경험이 있는지
3. 사전 상담에서 품목/등급 및 성적서 범위를 같이 논의할 수 있는지
4. 기관 양식이 명확한지

## 한 줄 정리

기관 이름보다 먼저 중요한 것은, **우리 제품 범위에 맞는 시험을 실제로 해줄 수 있는지와 사전 상담이 가능한지**입니다.
