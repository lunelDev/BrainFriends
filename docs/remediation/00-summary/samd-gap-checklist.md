# BrainFriends SaMD 준비 부족 항목 체크리스트

## 목적

이 문서는 현재 프로젝트 기준으로 공인성적서 대응 구조는 어느 정도 구현되었지만, 실제 SaMD 준비 관점에서 아직 부족한 항목을 정리한 체크리스트입니다.

## 1. 이미 확보된 항목

### 프로젝트 내부 구조

- SW V&V 추적성 구조
- deterministic SW V&V 시험 및 날짜별 실행 로그
- 브라우저 저장 최소화 및 보안 정책 문서
- measured-only AI 평가셋 분리
- AI 평가셋 DB 저장 및 버전 비교 모니터링
- 치료사 시스템 화면에서 V&V / 보안 / AI 평가 상태 확인 가능

### 제출 준비용 내부 문서

- 제품 정의 1장
- SW V&V 제출 개요 및 현재 운영본 결과서
- 사이버보안 저장 현황 및 최종 고정 보고서
- AI 성능평가 제출 개요 및 현재 운영본 보고서
- 시험기관 사전 문의 1페이지 요약본

## 2. 아직 부족한 항목

### A. 품목 / 등급 확정

- 현재 사용목적 문구 기준으로 실제 품목과 등급을 확정해야 함
- 이 항목이 확정되어야 필요한 성적서 종류와 심사 범위를 정확히 정할 수 있음

### B. 시험기관 요구 형식 확정

- 현재 문서는 내부 기준 문서 성격이 강함
- 실제 시험기관이 요구하는 표, 결과서, 부록, 증적 형식으로 다시 맞춰야 함

### C. 품질관리기준(GMP) 제출 묶음

- 조직/품질 문서
- 문서관리 및 변경관리 절차
- 완제품 시험 절차서 및 시험성적서
- 전자적 침해행위 보안 관련 절차서

즉, 코드 외의 품질시스템 문서 묶음이 더 필요함

### D. 사용적합성 자료

- 사용자 화면의 사용 흐름이 실제 사용 환경에서 적절한지 확인하는 자료
- 사용 오류 가능성, 주요 사용 시나리오, 위험 완화 설계 자료가 필요할 수 있음

### E. AI 성능평가 프로토콜 보강

- transcript / 자음 / 모음 / 안면 기준 라벨 정의
- 평가셋 포함 / 제외 규칙의 외부 설명 수준 정리
- 오류 사례 부록 누적
- 시험기관이 요구하는 지표 형식에 맞춘 결과표 정리

### F. 실제 시험 수행 및 성적서 수령

- 현재 구조와 문서는 시험을 받을 준비 수준
- 아직 실제 시험기관 시험 수행 및 성적서 발급 단계는 아님

## 3. 지금 가장 먼저 해야 할 것

1. 시험기관 사전 문의
2. 품목 / 등급 확인
3. 필요한 성적서 종류 확인
4. 시험기관 양식에 맞춘 문서 포맷 조정
5. 실제 시험 진행

## 4. 현재 판단

현재 프로젝트는 **SaMD를 위한 공인성적서 준비 구조는 갖춘 상태**로 볼 수 있습니다.

다만 아직은 다음 단계가 남아 있습니다.

- 외부 규제 기준에 맞춘 문서 재편
- 시험기관 요구 범위 확인
- GMP / 사용적합성 / AI 프로토콜 보강
- 실제 시험 및 성적서 확보

## 5. 한 줄 정리

현재 BrainFriends는 **“공인성적서를 받을 수 있도록 내부 구조를 정리한 상태”**이며, **“실제 SaMD 제출을 위한 외부 심사/시험 준비”**는 아직 남아 있습니다.
