# AI 성능평가 오류 사례 기록서

## 목적

이 문서는 measured-only 평가셋 운영 과정에서 제출 또는 내부 검토에 활용할 오류 사례를 누적 기록하기 위한 문서입니다.

## 기록 기준

- transcript 누락
- tracking 품질 저하
- processing 지연
- 발음/자모 점수 이상치
- 버전 전환 후 성능 delta 이상치

## 현재 상태

- 현재 운영본 기준 별도 오류 사례 부록은 아직 누적 중
- 오류 사례는 `/therapist/system/evaluation` 화면과 `ai_evaluation_samples` 비교 기준으로 추가 기록
