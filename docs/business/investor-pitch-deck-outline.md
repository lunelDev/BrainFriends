### 페이지 5 — 제품 시연

**핵심 메시지**
> 환자는 3스텝으로 훈련 완료. 치료사는 콘솔에서 환자 진도·음향 지표를 한눈에 본다. 데이터 파이프라인은 전 구간 암호화·감사로그 적용.

**필요 자료**
- **환자 플로우 3스텝**: 로그인 → 훈련 세션 → 결과 리뷰 (스크린샷 3컷)
- **치료사 콘솔**: 환자 목록 · 진도 대시보드 · 알람 (스크린샷 1~2컷)
- **데이터 파이프라인 다이어그램**: Device → FastAPI → Parselmouth/Whisper/MediaPipe → DB → 치료사 뷰
- 실시연 가능하면 QR코드로 데모 영상 링크 추가

### 페이지 6 — 기술 차별화

**핵심 메시지**
> 결정론적 음향 분석(Parselmouth) + K-WAB 기반 점수화 + 멀티모달(음성·얼굴) — 허가 심사에 필요한 "설명가능·재현가능" 기술 스택.

**필요 자료**
- 차별화 4축:
  1. Parselmouth 기반 결정론적 음향 파라미터 (REQ-ACOUSTIC-001~012)
  2. K-WAB(한국판 웨스턴 실어증 검사) 기반 점수화 체계
  3. 멀티모달 — 음성만이 아닌 얼굴 표정·조음 움직임까지
  4. 공인 증빙 가능한 구조 (재현성·감사추적성)
- 경쟁 제품 대비 기술 차별화 표 (글로벌 Top 2 대비)

### 페이지 9 — 비즈니스 모델

**핵심 메시지**
> DTx 처방 기반 유통 + 보험 수가 + 병원 구독 모델. B2B2C로 확장하여 환자 접근성·수익성 동시 확보.

**필요 자료**
- 수익 구조 4가지:
  1. **DTx 처방 유통** — 의사 처방 → 환자 결제 or 보험
  2. **보험 수가** — 급여 편입 시 공단 수입
  3. **병원 구독 모델** — 병원당 월정액 라이선스
  4. **B2B2C 확장** — 복지관·요양시설·재활병원 B2B + 개인 환자 C
- 예상 ARPU·ARPPU (가정 기반, 출처 명시)
- 시장 진입 순서: 병원 B2B → 수가 편입 → 직접 B2C
