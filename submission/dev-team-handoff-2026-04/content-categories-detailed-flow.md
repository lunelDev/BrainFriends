# 환자가 화면에서 어떻게 사용하나요

웹 브라우저 / 모바일 / 마이크 · 카메라만 있으면 진행

---

## 1. 자가진단 — 약 15-20분, 1회 측정

### 진행 흐름

```
장소 카드 6개 중 1개 선택  →  6단계 미니 검사 자동 진행  →  종합 점수 + 리포트
```

### 6단계 환자 동작

| 단계 | 환자 동작 |
| --- | --- |
| 1 · 듣고 답하기 | 음성 듣고 ⭕ / ❌ 누르기 |
| 2 · 따라 말하기 | 화면 문장을 마이크로 따라 말하기 |
| 3 · 이름 부르기 | 그림 보고 사물 이름 말하기 (🍎 → "사과") |
| 4 · 자유롭게 말하기 | 주제어 보고 막힘 없이 이야기 |
| 5 · 소리내어 읽기 | 화면 글을 또박또박 읽기 |
| 6 · 자판 입력 | 화면 단어를 키보드로 입력 |

---

## 2. 언어재활 — 반복 훈련

### 진행 흐름

```
6단계 영역 메뉴 진입  →  단계 선택 또는 순차 진행  →  매 회기 결과 누적  →  회복 흐름 그래프
```

### 자가진단과 다른 점

- **횟수**: 자가진단은 1회 측정 / 언어재활은 의료진 권장 일정에 따라 반복
- **결과**: 자가진단은 종합 리포트 1장 / 언어재활은 매 회기 누적 + 진행률 그래프
- **의료진 모니터링**: 의사·치료사가 결과를 함께 확인할 수 있는 기능 포함

### 6단계 영역

자가진단과 동일한 6단계 (듣고 답하기 / 따라 말하기 / 이름 부르기 / 자유롭게 말하기 / 소리내어 읽기 / 자판 입력) 를 반복 훈련 형태로 진행.

---

## 3. 노래방 훈련 — 약 1분 / 곡

### 진행 흐름

```
곡 선택  →  가사 + 음정 가이드 표시  →  마이크로 따라 부르기  →  곡 점수 + 결과
```

### 화면에 보이는 것

- **위쪽**: 흘러가는 가사 띠
- **가운데**: 음정 가이드 + 내 음정 위치 (실시간 일치도)
- **아래**: 마이크 입력 표시 + 진행 시간 바

## 사용 환경 (3개 카테고리 공통)

- **장소**: 의료기관 또는 자택
- **기기**: 일반 PC · 태블릿 · 스마트폰 (특수 장비 없음)
- **필요한 것**: 인터넷, 마이크, 카메라
